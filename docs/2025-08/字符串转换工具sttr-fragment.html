案例一：DevOps 工程师在 CI/CD 中的应用

在 Jenkinsfile 或 .gitlab-ci.yml 中，经常需要对构建产物进行哈希以生成校验和。

# 旧方法 (依赖特定环境)
# checksum=$(sha256sum my-app.jar | awk '{print $1}')

# sttr 方法 (跨平台且简洁)

使用 sttr 后，CI/CD 脚本不再需要担心运行代理是 Windows 还是 Linux，降低了脚本的复杂性和维护成本。
